# Use
1. input your code anywhere.
2. run it.
    ``` shell
    python main.py (-f [filename]) (-s [display spend])
    ```
    `[filename]` defaults to `main.txt`.  
    `[display spend]` defaults to `1`.  
    The larger the `[display spend]`, the slower it will be displayed.
